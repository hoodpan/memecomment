

# Memecomment







## Install
```bush
// install dependencies
npm install
```
## Run
### Development
```bush
npm run dev
```
### Production(Build)
```bush
npm run build
```

## 简介
&emsp;&emsp;Memecomment(网络新词百科)是基于Vue.js，搭配使用iview-admin形成的一个前端项目，通过搜罗网络新词，帮助用户在需要的时候快速的查询到其正确的释义及渊源。

## 功能


- 词条搜索
    - 相关推荐
    - 随机刷新
- 分类词条
- 热词推荐
- 贡献词条

- 错误页面
    - 403页面
    - 404页面
    - 500页面

- 换肤
- 收缩侧边栏
- tag标签导航
- 面包屑导航
- 全屏/退出全屏


## 文件结构
```shell
.
├── build  项目构建配置
├── config  开发相关配置
├── public  打包所需静态资源
└── src
    ├── api  AJAX请求
    └── assets  项目静态资源
        ├── icons  自定义图标资源
        └── images  图片资源
    ├── components  业务组件
    ├── config  项目运行配置
    ├── directive  自定义指令
    ├── libs  封装工具函数
    ├── locale  多语言文件
    ├── mock  mock模拟数据
    ├── router  路由配置
    ├── store  Vuex配置
    ├── view  页面文件
    └── tests  测试相关
```
